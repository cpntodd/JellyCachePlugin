Jelly Cache Manager

Cache Directory Overview

This plugin monitors and manages Jellyfin cache directories automatically.

Category Path Max Size Status
Cache /var/cache/jellyfin 1000 MB Monitored
Image Cache /var/cache/jellyfin/images 500 MB Monitored
Program Data /var/lib/jellyfin 2000 MB Monitored
Logs /var/log/jellyfin 100 MB Monitored
Metadata /var/lib/jellyfin/metadata 1500 MB Monitored
Transcodes /var/cache/jellyfin/transcodes 2000 MB Monitored
Web /usr/share/jellyfin/web 500 MB Monitored

Configuration

Cleanup runs every 24 hours. Alerts enabled with 80% threshold.

All settings are configurable in the plugin configuration.

Manual Cleanup

Manual cleanup is performed automatically by the background service.

Logs

Check Jellyfin logs for cleanup activities.
